# PumpkinWorldEvents

## Install
- Put `PumpkinWorldEvents-1.0.jar` in your server `plugins/` folder.
- Restart the server.

## Usage
PumpkinWorldEvents introduces global events such as meteors, invasions, and boss encounters. Events trigger automatically or can be started manually by admins.

Players participate by fighting mobs, collecting loot, or completing objectives during events.

Events are broadcast to the entire server and can affect large areas.

## Commands & permissions
- `/event start <type>` — `pumpkin.events.admin`
- `/event stop` — `pumpkin.events.admin`
- `/event info` — `pumpkin.events.use`
