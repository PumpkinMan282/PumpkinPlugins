# PumpkinAbilities

## Install
- Put `PumpkinAbilities-1.0.jar` in your server `plugins/` folder.
- Restart the server.

## Usage
Players can select a hero/class that grants them a set of abilities. Abilities are triggered through actions such as right-clicking, attacking, or using specific items.

Abilities have cooldowns and can be upgraded to improve power, reduce cooldown, or add new effects. Players can chain abilities together to create combos for stronger outcomes.

Each hero has a unique playstyle, encouraging different builds and strategies.

## Commands & permissions
- `/ability select <hero>` — `pumpkin.abilities.use`
- `/ability upgrade <ability>` — `pumpkin.abilities.use`
- `/ability reset` — `pumpkin.abilities.admin`
