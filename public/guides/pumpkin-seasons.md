# PumpkinSeasons

## Install
- Put `PumpkinSeasons-1.0.jar` in your server `plugins/` folder.
- Restart the server.

## Usage
PumpkinSeasons cycles the world through seasonal changes such as spring, summer, fall, and winter.

Each season affects gameplay by modifying things like crop growth, player effects, and environmental conditions.

Seasons change automatically over time or can be set manually.

## Commands & permissions
- `/season` — `pumpkin.seasons.use`
- `/season set <season>` — `pumpkin.seasons.admin`
- `/season reload` — `pumpkin.seasons.admin`
